# AI Chat UI 设计文档

**日期**: 2026-04-23  
**主题**: 现代极简风格 UI 改造  
**风格**: Notion 风格 + 暖白配色

---

## 1. 设计概述

### 1.1 设计目标
- 提升视觉美感，采用现代极简风格
- 优化用户体验，减少视觉噪音
- 保持功能完整性，不破坏现有交互
- 使用暖白配色，长时间使用不疲劳

### 1.2 设计原则
- **大量留白**: 让内容呼吸，减少压迫感
- **层次清晰**: 通过背景色区分功能区域
- **极细边框**: 提供结构感但不突兀
- **悬停反馈**: 操作元素在悬停时才显示，减少干扰

## 5. 交互细节

### 5.1 过渡动画

**通用过渡**:
- 时长: 150ms
- 缓动: ease-out
- 属性: background-color, border-color, color, opacity, transform

**消息气泡操作按钮**:
- 动画: fadeIn
- 时长: 150ms
- 从: opacity 0, translateY(-4px)
- 到: opacity 1, translateY(0)

**加载动画**:
- 类型: bounce
- 时长: 1s
- 循环: infinite

### 5.2 悬停状态

**按钮悬停**:
- 背景色变深
- 边框色变深
- 过渡: 150ms ease-out

**列表项悬停**:
- 背景色变化
- 删除按钮显示
- 过渡: 150ms ease-out

**消息气泡悬停**:
- 操作按钮显示
- 过渡: 150ms ease-out

### 5.3 聚焦状态

**输入框聚焦**:
- 边框: 1px solid `#d1d5db`
- 背景: `#faf9f7`
- outline: none

**按钮聚焦**:
- outline: 2px solid `#2563eb`
- outline-offset: 2px

## 6. 响应式设计

### 6.1 断点

- **桌面**: > 1024px（默认布局）
- **平板**: 768px - 1024px（侧边栏可折叠）
- **手机**: < 768px（侧边栏隐藏，汉堡菜单）

### 6.2 移动端适配

**侧边栏**:
- 默认隐藏
- 汉堡菜单触发
- 全屏覆盖显示
- 背景遮罩: rgba(0, 0, 0, 0.5)

**消息气泡**:
- 最大宽度: 85%（更宽，充分利用屏幕）

**输入框**:
- 固定在底部
- 安全区域适配（iPhone 底部横条）

### 8.2 CSS 变量定义

```css
:root {
  /* 背景 */
  --bg-page: #faf9f7;
  --bg-sidebar: #f5f4f2;
  --bg-card: #ffffff;
  --bg-input: #f5f4f2;
  --bg-code: #f8f9fa;
  --bg-code-header: #f0eeeb;
  
  /* 文字 */
  --text-primary: #1a1a1a;
  --text-secondary: #6b6b6b;
  --text-tertiary: #9ca3af;
  --text-sidebar: #374151;
  
  /* 强调色 */
  --accent-primary: #2563eb;
  --accent-hover: #1d4ed8;
  --accent-user: #e8e4f9;
  --accent-success: #10b981;
  --accent-error: #ef4444;
  
  /* 边框 */
  --border-default: #e8e6e3;
  --border-hover: #d1d5db;
  --border-divider: #f0eeeb;
  
  /* 圆角 */
  --radius-sm: 4px;
  --radius-md: 8px;
  --radius-lg: 12px;
  --radius-xl: 16px;
  
  /* 过渡 */
  --transition-fast: 150ms ease-out;
}
```

### 8.3 注意事项

1. **保持功能完整**: 所有现有功能（编辑、删除、导出、搜索等）必须保留
2. **渐进式改造**: 可以先改核心组件（Sidebar、ChatMessage、InputBox），其他逐步迭代
3. **测试交互**: 特别关注悬停显示的操作按钮，确保在各种情况下都能正常显示
4. **性能考虑**: 动画使用 CSS transition，避免使用 JavaScript 动画
5. **可访问性**: 确保颜色对比度符合 WCAG 标准，键盘导航正常
